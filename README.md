# oh-my-elvish
## Installation
``` shell
use epm
epm:install github.com/muesli/elvish-libs
git clone https://github.com/darcy-shen/oh-my-elvish.git
mv ~/.config/elvish ~/.config/elvish_bak
mv oh-my-elvish ~/.config/elvish
```

## Get Elvish
